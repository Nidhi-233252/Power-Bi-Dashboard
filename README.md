# Car Sales Dashboard 
This dashboard provides a comprehensive analysis of car sales data, helping users understand vehicle pricing trends, mileage patterns, fuel type distribution, and manufacturing year insights. The dashboard was developed using Power BI and includes data cleaning, KPI tracking, and interactive visualizations.

# Key Performance Indicators (KPIs)
- Total Cars Available
- Average Selling Price
- Average Mileage
- Average Engine Size
# Dashboard Features
- Average Price by Model analysis
- Fuel Type Distribution visualization
- Cars by Year of Manufacture trend analysis
- Mileage vs Price correlation using a Scatter Plot
- Interactive slicers for Manufacturer, Fuel Type, and Year of Manufacture
# Business Insights
- Identify the most valuable car models.
- Analyze the impact of mileage on vehicle price.
- Understand the distribution of different fuel types.
- Track the availability of vehicles across manufacturing years.
- Support data-driven decisions for buyers, sellers, and dealerships.
# Tools & Technologies
- Power BI Desktop
- Power Query for Data Cleaning
- DAX Measures for KPI calculations
- Interactive Dashboard Design

This dashboard demonstrates data visualization, business intelligence, and analytical skills by transforming raw car sales data into actionable insights.
